# Algorithms

A collection of algorithm implementations in **C#**, useful for practicing data structures and algorithm design. Each algorithm is implemented with clean, readable code and meaningful structure.

## 📌 Contents

This repository currently includes:

- 📊 Sorting Algorithms  
  - Bubble Sort  
  - Selection Sort  
  - Insertion Sort  
  - Merge Sort  
  - Quick Sort  

- 🔍 Search Algorithms  
  - Linear Search  
  - Binary Search  

- 📐 Math Algorithms  
  - Prime Number Check  
  - Factorial Calculation  
  - Fibonacci Series  

- 💡 Miscellaneous  
  - Palindrome Checker  
  - String Reversal  
  - Swap Numbers  

## 🛠️ Technologies Used

- C#
- .NET 8 SDK
- Console Application Projects
